# Topology Optimization for 3D Structures Design

This is a portfolio case study based on my MSc dissertation project at Cardiff University. The project explores how geometry and loading conditions affect topology-optimized 3D structures.

## Project Overview

The objective was to investigate lightweight structural design using topology optimization. The study used the SIMP method, finite element analysis, and MATLAB-based `top3d.m` simulations to optimize material distribution under different design conditions.

## Tools and Methods

- MATLAB
- top3d.m
- SIMP method
- Finite Element Analysis
- 3D structural optimization
- Compliance minimization
- Volume fraction constraint: 30%

## Case Studies

### Case A: Geometry Variation

Three aspect ratios were tested to understand how the design domain affects the final optimized structure:

- A1: 3:1:0.5
- A2: 2:2:1
- A3: 1:3:1

### Case B: Loading Variation

Four loading conditions were studied:

- Central point load
- Uniform load
- Triangular distributed load
- Edge-concentrated load

## Key Outcome

The project showed that elongated structures aligned with the main load path achieved better stiffness-to-weight performance, while loading type strongly influenced the final topology and material distribution.
